                             SBM Joke Demo
                           =================

This directory contains a demo example for the SmartBody Module (SBM),
first presented at AAMAS 2008.  It relies on the assets in sbm-common
and a couple of sequences in sbm-test.

data/sbm-demo-joke  - This directory
  scripts/          - demo-joke sequences (.seq files)
  bml/              - BML used in the demo

SmartBody was built by University of Southern California's Institute for
Creative Technologies and Information Sciences Institute.
