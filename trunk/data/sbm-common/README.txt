                         SBM Common Data
                       ===================

This directory contains assets and sequences common to many projects using
the SmartBody Module (SBM).

data/sbm-common     - This directory
  common-sk/        - Test/Experimental Assets for common.sk skeleton
    common.sk       - Common skeleton
    geom/           - "Doctor" rigid-body geometry for the internal viewer
	motion/         - .skm formatted animations
	pose/           - .skp formatted poses
  scripts/          - Shared Sequences (.seq files)
    common-init.seq - Script to load the common assets.

SmartBody was built by University of Southern California's Institute for
Creative Technologies and Information Sciences Institute.
