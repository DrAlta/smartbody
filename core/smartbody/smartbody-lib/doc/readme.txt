==== SOURCE FILE DOCUMENTATION ====

  Documentation of the source code can be generated
  with doxygen:
  http://www.stack.nl/~dimitri/doxygen/

  Doxygen binaries can be downloaded from here:
  http://www.stack.nl/~dimitri/doxygen/download.html

  Once doxygen binary is downloaded, it can be used
  with the provided configuration file doxyfilewin.txt,
  by calling *from inside the doc folder* :
  doxygen doxygen_config.txt
